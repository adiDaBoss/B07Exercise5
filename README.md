# B07Exercise5

This project solves the question given in Tutorial Exercise 5 by group 6.

# File Structure

The files are organised into `src/` which consists of the the main project files.

`src/main/java` Consists of the package `exercise5`. This package consists of the shapes Triangle, Point, Circle, Rectangle.

`src/test/java/exercise5` consists the test cases that run for each of the Shapes and their respective functions.

`pom.xml` is a file created by Maven and consists details about the builds and other project configurations.
